Il progetto sull'analisi dei reclami dei consumatori è un semplice file excel dove ho lavorato su un database che andava a mostrare le lamentele che si presentavano con maggior frequenza all'interno dei vari stati, utilizzando quindi funzioni condizionali e altre funzioni basilari di EXCEL. 
Il progetto sulle analisi analisi delle operazioni di trading invece è un attimino più articolato. Mi sono servito dell'utilizzo di tabelle pivot, tabelle di contingenza, funzioni di manipolazione del testo e operatori condizionali. 
Il progetto sui punti vendita di una catena di distribuzione di elettronica invece è stato fatto utilizzando un data set più articolato. Ho unito file CSV che indicavano le vendite mensili di un'azienda, e dei file excel che parlavano dei prodotti e i vari negozi con i rispettivi responsabili. Ho lavorato attraverso la PowerQuery pulendo e ordinando i dati in modo da rendere il report molto più semplice da costruire. Mi sono servito delle funzioni fondamentali di merge dopo aver pulito il dataset, poi ho effettuato i vari collegamenti creando anche misure necessarie al fine di rendere i calcoli narrabili attraverso il report. Ho creato diverse pagine con più visual per avere una narrazione graficamente chiara e leggibile.
